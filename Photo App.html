Photo App

Tags:

HTML/CSS

JavaScript

Flask

React

Description:

We wanted to make a website that replicated the features of Instagram. We began by making the structure of the website with HTML, CSS and JavaScript.

The website was later implemented with React to abstract logical groupings/components of code.

Then, we used Flask, a python web framework, in order to create REST APIs, generate server-side templates and interact with databases.

This allowed us to make the website interactive. For example, users could like/unlike photos, write comments and follow/unfollow different users.

We also utilized JSON web tokens to authenticate users and protect against cross-site request forgery attacks.
